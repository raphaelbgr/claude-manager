API Endpoints

  • GET
    /health
    Server health + scan metadata
  • GET
    /api/sessions
    All sessions grouped by machine & project
  • GET
    /api/sessions/{machine}
    Sessions for a specific machine
  • POST
    /api/sessions/scan
    Force an immediate rescan
  • POST
    /api/sessions/launch
    Launch a session in a terminal (Phase 2)
  • GET
    /api/fleet
    Fleet machine health status
  • GET
    /api/tmux
    Tmux sessions (Phase 2)
  • WS
    /ws
    WebSocket: subscribe to sessions / fleet / tmux channels
